# 💱 Currency Converter

A simple and responsive **currency converter** web application built using **HTML**, **CSS**, and **JavaScript**.

🌐 **Live Demo:** [https://thilips-currency-converter.netlify.app](https://thilips-currency-converter.netlify.app)

---

## 📌 Features

- 🔁 Real-time currency conversion
- 🌍 Supports multiple international currencies
- 📱 Responsive design (works well on desktop and mobile)
- 💡 Easy-to-use and lightweight interface

---

## 🛠️ Built With

- **HTML5** – for the structure  
- **CSS3** – for styling and layout  
- **JavaScript (Vanilla)** – for fetching API data and conversion logic
